# CASBOT2 ROS 2 二次开发手册（发布版）

## 概览

CASBOT2 二次开发主要通过 ROS 2 的 Topic / Service / Action 与机器人交互。

## 核心接口

- 语音服务：`/voice_svr` (`crb_ros_msg/srv/Voice`)
- 技能触发：`/casbot/event_service` (`crb_ros_msg/srv/ActionEvent`)
- 导航速度：`/navigation/cmd_vel` (`geometry_msgs/msg/Twist`)
- 上身调试：`/motion/upper_body_debug` + `/upper_body_debug/joint_cmd`
- 全身调试：`/motion/whole_body_debug` + `/motion/joint_cmd`

## 运控关键说明

- `/navigation/cmd_vel` 当前仅使用 `linear.x` 与 `angular.z`，`linear.y` 未接入。
- 全身调试输入与反馈接口分别为：
  - `/motion/joint_cmd` (`crb_ros_msg/msg/JointStateData`)
  - `/motion/joint_state` (`crb_ros_msg/msg/JointStateData`)
- `debug_mode_params.use_default_kp_kd=false` 时，`kp/kd` 需随消息下发。

## 建议流程

1. 上电后先确认模式状态；
2. 进入对应调试模式再发布控制指令；
3. 低速、低增益开始联调；
4. 使用 `/motion/joint_state` 与 `/joint_states` 双通道观测。

## 说明

本发布版文档是对外说明的精简版。若需更完整细节，可结合项目内部文档继续扩展。

# CASBOT2 ROS 2 Secondary Development Manual (Release Edition)

## Overview

CASBOT2 secondary development mainly interacts with the robot through ROS 2 Topics, Services, and Actions.

## Core Interfaces

- Voice service: `/voice_svr` (`crb_ros_msg/srv/Voice`)
- Skill trigger: `/casbot/event_service` (`crb_ros_msg/srv/ActionEvent`)
- Navigation velocity: `/navigation/cmd_vel` (`geometry_msgs/msg/Twist`)
- Upper-body debug: `/motion/upper_body_debug` + `/upper_body_debug/joint_cmd`
- Whole-body debug: `/motion/whole_body_debug` + `/motion/joint_cmd`

## Motion Control Notes

- `/navigation/cmd_vel` currently uses only `linear.x` and `angular.z`; `linear.y` is not connected.
- Whole-body debug input/output interfaces:
  - `/motion/joint_cmd` (`crb_ros_msg/msg/JointStateData`)
  - `/motion/joint_state` (`crb_ros_msg/msg/JointStateData`)
- When `debug_mode_params.use_default_kp_kd=false`, `kp/kd` must be included in command messages.

## Recommended Workflow

1. Confirm robot mode after power-on.
2. Enter the corresponding debug mode before sending control commands.
3. Start with low speed and low gains.
4. Observe both `/motion/joint_state` and `/joint_states`.
